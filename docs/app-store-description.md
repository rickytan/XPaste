## App Name

## Subtitle (30 chars max)

## Description

**Boost your IDE productivity with XCPaste!**

XCPaste is a powerful Source Editor Extension that transforms how you paste text into your code. Save time and write cleaner code with these essential features:

### Features

**📋 Paste as Base64 String**
Instantly convert any text—or even image data from your clipboard—into a Base64 encoded string. Perfect for embedding data directly in your code.

**📝 Paste as String Literal**
Automatically escape quotes, newlines, and special characters. Paste HTML, JSON, or any multi-line text directly as a valid Objective-C or Swift string literal without manual editing.

**🌐 Paste as Unicode String**
Convert non-ASCII characters (including emojis) to their Unicode escape sequences. Great for internationalized apps or when you need precise character representation.

**🔗 Paste as URL Encoded String**
Transform URLs with Chinese or special characters into properly encoded strings. No more manual URL encoding!

**📦 Paste JSON as Objective-C Objects**
Convert JSON strings directly into Objective-C dictionary, array, or string literals. Paste API responses as ready-to-use code instantly.

**⚡ Sort Imports**
Automatically organize your import statements. Group framework imports (`<>`) and local imports (`""`), then sort alphabetically. Works with both Objective-C (`#import`) and Swift (`import`).

---

**Installation:**
1. Open XCPaste.app
2. Click "Enable XCPaste"
3. Ensure "XCPaste Source Editor" is checked in System Extensions
4. Relaunch your IDE — find XCPaste in the Editor menu

**Customize keyboard shortcuts:**
Open Preferences → Key Bindings → search "xcpaste" → assign your favorite shortcuts.

---

**Requirements:**
- macOS 10.13 or later
- Xcode 12 or later
